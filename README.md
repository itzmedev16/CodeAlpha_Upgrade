Social Media Platform

Features
- User Profiles
- Create & Manage Posts
- Comment System
- Like & Follow Functionality

Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Express.js (Node.js)
- Database: MongoDB

How to Run
1. Clone the repository
2. Run: npm install
3. Start server: npm start
4. Open frontend in browser

 Description
This is a mini social media platform where users can create profiles, share posts, comment, like posts, and follow other users. The project demonstrates full-stack development using modern web technologies.
